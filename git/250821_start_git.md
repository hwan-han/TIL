# git command

### 필수 command
git add filename  
git commit  
git push origin main  
=> git(로컬)에서 github으로 적용

### 현업 prefix
feat : 기능 개발 관련  
fix : 오류 개선 혹은 버그 패치  
docs : 문서화 작업  
test : test 관련  
conf : 환경설정 관련  
build : 빌드 관련 작업  
ci : Continuous Integration 관련  
chore : 패키지 매니저, 스크립트 등  
style : 코드 포매팅 관련
